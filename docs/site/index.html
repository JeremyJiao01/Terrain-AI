Quick Start

Two steps to supercharge your AI assistant.

Step 1: Install & Init

npx code-graph-builder@latest --setup

Step 2: Configure MCP

Add to your Claude Code / Cursor / Windsurf settings:

{
  "mcpServers": {
    "code-graph-builder": {
      "command": "npx",
      "args": ["-y", "code-graph-builder@latest", "--server"]
    }
  }
}
