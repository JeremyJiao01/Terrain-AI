官网形态

选择最符合你预期的方向

A

静态展示型

精美的 Landing Page，动态展示调用链图谱、代码搜索效果截图/动画，无后端，部署在 GitHub Pages

优点

  • 零成本托管
  • 加载极快
  • 维护简单

限制

  • 交互为预设动画
  • 无真实数据
B

前端交互型（纯客户端）

页面内嵌一个预构建的代码图谱数据集（JSON），访客可以真正搜索函数、点击查看调用链，所有运算在浏览器完成

优点

  • 真实交互体验
  • 仍然零后端
  • 可 GitHub Pages 部署

限制

  • 需预先生成数据集
  • 数据量有上限
C

在线 Playground（有后端）

访客可以粘贴一段代码或填入 GitHub 仓库地址，后端实时分析并返回图谱结果

优点

  • 最震撼的 Aha Moment
  • 真实演示产品能力

限制

  • 需要服务器
  • 运营成本
  • 开发量大
